# 💕 新年表白特效 - 使用说明

## 🚀 快速开始

### 方法1：在VS Code/Cursor中直接运行（推荐）

1. **安装Python扩展**（如果还没有）
   - 在VS Code/Cursor中按 `Cmd+Shift+X` (Mac) 或 `Ctrl+Shift+X` (Windows)
   - 搜索 "Python" 并安装 Microsoft 的 Python 扩展

2. **安装依赖库**
   - 打开终端（`Ctrl+` ` 或 `Cmd+` `）
   - 运行以下命令：
   ```bash
   pip install -r requirements.txt
   ```
   或者单独安装：
   ```bash
   pip install matplotlib numpy pillow
   ```

3. **运行脚本**
   - 打开 `表白特效.py` 文件
   - 点击右上角的 ▶️ 运行按钮
   - 或者按 `F5` 运行
   - 或者在终端运行：`python 表白特效.py`

4. **查看结果**
   - 脚本会自动在当前目录生成多张精美的图片
   - 所有图片都是高清PNG格式（300 DPI）

### 方法2：在终端中运行

```bash
# 1. 安装依赖
pip install matplotlib numpy pillow

# 2. 运行脚本
python 表白特效.py
```

## 💡 常见问题

### Q: 提示找不到matplotlib？
A: 运行 `pip install matplotlib numpy pillow` 安装依赖

### Q: 中文显示为方块？
A: 脚本已设置中文字体，如果还是显示不正常，可以修改代码中的字体设置

### Q: 动画生成失败？
A: 确保安装了pillow库：`pip install pillow`

### Q: 如何在VS Code中运行？
A: 
1. 确保安装了Python扩展
2. 打开 `表白特效.py`
3. 点击右上角运行按钮或按 `F5`
